# Personal Finance Manager
A comprehensive Java-based console application for managing personal finances, tracking expenses, and budgeting.
## Overview
The Personal Finance Manager is a robust console application that helps users track their income, expenses, and budgets. It provides detailed financial analytics and helps users maintain healthy financial habits through proper budgeting and spending analysis.
## Features
### Transaction Management
- Add income and expense transactions
- View all transactions with detailed information
- Search transactions by category or description
- Delete transactions
- View current balance summary
### Financial Analytics
- Spending analysis by category
- Monthly income vs expense summaries
- Cash flow analysis with savings rate calculation
- Visual spending patterns
### Budget Planning
- Set monthly budget limits for categories
- Track budget vs actual spending
- Receive overspending alerts
- Budget usage percentage tracking
### Security Features
- Local data encryption
- Input validation and error handling
- Secure file storage
## Technologies & Tools Used
- **Programming Language:** Java 11+
- **IDE:** Visual Studio Code
- **Extensions:** Java Extension Pack
- **Data Storage:** Encrypted text files
- **Libraries:** Java Standard Library
## Installation & Setup for VS Code
### Prerequisites
- Java JDK 11 or higher
- Visual Studio Code
- Java Extension Pack
Step-by-Step Installation
1. **Install Visual Studio Code**
   - Download from [code.visualstudio.com](https://code.visualstudio.com/)
   - Install the Java Extension Pack
